Welcome to my React training repository! This repository documents my daily assignments and exercises as I progress through various React concepts and challenges.

## 🛠️ Technologies Used

- React (v18+)
- JSX & ES6+ JavaScript
- React Router
- React Hooks (useState, useEffect, useContext, etc.)
- Context API & Redux (for state management)
- Axios (for data fetching)
- Styled Components / CSS Modules / Tailwind CSS
- Vite / Create React App

## 🧠 Recommended GitHub Repositories

To further enhance your React skills, explore these curated GitHub repositories:

- [30 Days of React](https://github.com/Asabeneh/30-Days-Of-React): A comprehensive 30-day challenge to learn React.
- [React Bits](https://github.com/vasanthk/react-bits): A collection of React patterns, techniques, tips, and tricks.
- [React in Patterns](https://github.com/krasimir/react-in-patterns): A free book discussing design patterns and techniques in React.
- [Real World React Apps](https://github.com/jeromedalbert/real-world-react-apps): A showcase of real-life React apps and their open-source codebases.
- [React Redux Links](https://github.com/markerikson/react-redux-links): A curated list of tutorials and resources on React and Redux.
